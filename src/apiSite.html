Cocktail API

This API allows the user to access a database of cocktail recipes. You can filter results by cocktail name, ingredients, glass type and whether or not the cocktail contains alcohol.

Endpoints

/v1/cocktail-recipes

Retrieves cocktail recipes from the database that match the parameters you have provided - or all cocktails if no parameters are added.

Parameters

Name - string (case-insensitive) - optional => Only cocktails with names including this string will be returned
Alcoholic - boolean - optional => If true, only cocktails with alcohol will be returned, if false, only cocktails without alcohol will be returned
Glass - string (case-insensitive) - optional => Only cocktails with glasses including this string will be returned
Ingredients - array - optional => Only cocktails with ingredients including all strings in array will be returned
MinIngredients - number - optional (default = 1) => Only cocktails with a minimum of this number of ingredients will be returned
MaxIngredients - number - optional (default = infinity) => Only cocktails with a maximum of this number of ingredients will be returned

Returns

JSON object containing all cocktails matching the specifications

    
/v1/cocktail-glasses

    
Retrieves all of the different glasses of the cocktails in the database.

    
Parameters

    
None

    
Returns

    
Array containing all glasses in database

    

    
/v1/cocktail-ingredients

    
Retrieves all of the different ingredients of the cocktails in the database.

    
Parameters

    
None

    
Returns

    
Array containing ingredients in database
